Basic Usage

Markup

    <div id="sliding-content">
      <ul>
        <li>item 1</li>
        <li>item 2</li>
        <li>item 3</li>
        <li>item 4</li>
      </ul>         
    </div>

Code

    $('#sliding-content').sliding();

Examples

Simple sliding horizontal

      $('.sliding-horizontal').sliding({
            mode: 'horizontal',
            items: 3,
            target: '.ui-sliding-navigation'
        });
